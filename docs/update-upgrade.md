# 更新与升级

---

## 插件更新

Obsidian 社区插件浏览器自动更新。BRAT 安装用户同样自动检测更新。

## Python 包更新

```bash
paperforge update
# 或
pip install --upgrade paperforge
```

## 备份

更新前建议备份：
- `.env`（API key）
- `Resources/` 目录（文献笔记）
- `System/PaperForge/ocr/`（OCR 结果）
