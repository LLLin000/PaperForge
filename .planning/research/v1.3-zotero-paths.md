## 3. Path Format Analysis

### Real path format from BBT export

```
D:\L\Med\Research\99_System\Zotero\storage\<8位代码>\<原始文件名>.pdf
```

**Components:**
- `D:\L\Med\Research\99_System\Zotero\storage\` — Zotero 数据目录 + storage
- `<8位代码>` — Zotero item key (如 `RU33UHK8`, `JY4AYIWW`)
- `<原始文件名>` — 导入时的原始 PDF 文件名（可能含中文、特殊字符）

### Current code expectations (v1.2)

| 格式类型      | 示例                              | 来源         |
| ------------ | ---------------------------------- | ----------- |
| `storage:`前缀 | `storage:KEY/KEY.pdf`              | BBT 配置输出 |
| 裸相对路径    | `KEY/KEY.pdf`                      | BBT 默认输出 |
| 绝对路径      | `D:\...\Zotero\storage\KEY\...pdf` | **真实导出** |

**发现：当前代码主要处理 `storage:` 前缀和裸路径，但真实 BBT JSON 导出使用的是绝对 Windows 路径。**

## 4. Key Fields for Path Resolution

| 字段           | 示例值                                        | 用途                   |
| ------------- | --------------------------------------------- | ---------------------- |
| `citationKey`   | `a.hamidPlateletrichPlasmaRotator2021`        | 引用键，用于笔记文件名  |
| `itemKey`       | `ME6BJZVS`                                    | 主条目 Zotero key       |
| `attachments[0].path` | `D:\...\storage\RU33UHK8\...pdf`      | **PDF 绝对路径**        |
| `attachments[0].uri` | `http://zotero.org/users/.../RU33UHK8` | 附件 Zotero key         |
| `attachments[0].select` | `zotero://select/library/items/RU33UHK8` | Zotero 选择链接       |

**重要：**
- `itemKey` (主条目) ≠ attachments 中的 key (附件条目)
- 附件有自己的 Zotero key（在 `uri` 和 `select` 中）
- 附件 key 与 storage 目录名一致（`RU33UHK8`）

## 6. Path Normalization Requirements for v1.3

### Input formats to support

1. **绝对路径**（当前真实格式）
   ```
   D:\L\Med\Research\99_System\Zotero\storage\RU33UHK8\文件名.pdf
   ```

2. **`storage:` 相对路径**（BBT 配置可输出）
   ```
   storage:RU33UHK8/文件名.pdf
   ```

3. **裸相对路径**（BBT 默认）
   ```
   RU33UHK8/文件名.pdf
   ```

4. **URI 格式**（从 `uri` 字段组装）
   ```
   zotero://select/library/items/RU33UHK8
   ```

### Output requirements

1. **Vault 相对路径**（用于 Wikilink）
   ```
   system/Zotero/storage/RU33UHK8/文件名.pdf
   ```

2. **Obsidian Wikilink**
   ```markdown
   [[system/Zotero/storage/RU33UHK8/文件名.pdf]]
   ```

3. **Windows 路径**（用于本地文件操作）
   ```
   D:\L\Med\Research\99_System\Zotero\storage\RU33UHK8\文件名.pdf
   ```

## 7. Wikilink Standard (监工指定)

**格式：** `[[相对路径]]`

**要求：**
- 不使用 Markdown 标准链接 — 使用 `[[wikilink]]` 而非 `label → url` 格式
- 使用 Obsidian wikilink 语法
- 路径相对于 Vault 根目录
- 支持双链图谱和反向链接

**示例：**
```markdown
PDF链接：[[system/Zotero/storage/RU33UHK8/文件名.pdf]]
笔记链接：[[Literature/骨科/ME6BJZVS - 论文标题]]
```

**路径分隔符：**
- Vault 内使用 `/`（即使 Windows 系统）
- Obsidian 会自动处理跨平台路径
